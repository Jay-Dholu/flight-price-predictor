About Flight Price Predictor

Welcome to Flight Price Predictor – a web application designed to help users estimate the cost of airline tickets using machine learning. This project aims to simplify your flight planning by offering real-time price predictions based on past data and key flight features.


How It Works

The model uses a dataset containing flight information like airline, source, destination, journey duration, and number of stops. I applied machine learning algorithms such as Random Forest and XGBoost to predict prices with a high degree of accuracy.

Technologies used in this project include Python, Pandas, Scikit-learn, Flask, and HTML/CSS for the frontend.
